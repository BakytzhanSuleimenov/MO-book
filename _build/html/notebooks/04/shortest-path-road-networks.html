Computing a real life shortest path#

During this course we make use of Jupyter notebooks hosted by Google Colab. Notebooks deployed on colab require neither python nor other dependencies to be installed on your own machine, you only need a browser (preferably chrome) and you may also need a google account if you want to execute them.

Notes:

  1. This notebook takes some extra trouble to install some packages, due to a more modern version of numpy and matplotlib being needed than the one installed by colab.

  2. From now on, we prepare our notebooks for colab since we assume that in case you prefer to run them locally then by know you know how and where to adapt them.

import subprocess
import sys

class ColabInstaller():

    def __init__(self):
        reqs = subprocess.check_output([sys.executable, '-m', 'pip', 'freeze'])
        self.installed_packages = [r.decode().split('==')[0] for r in reqs.split()]

    def on_colab(self):
        return "google.colab" in sys.modules

    def install(self, package):
        if self.on_colab():
            subprocess.check_call([sys.executable, "-m", "pip", "install", "-q", package])
            self.installed_packages.append(package)

    def upgrade(self, package):
        if self.on_colab():
            subprocess.check_call([sys.executable, "-m", "pip", "install", "-q", "--upgrade", package])

colab = ColabInstaller()
colab.upgrade('numpy')
colab.upgrade('matplotlib')
colab.install('geopandas')
colab.install('geopy')
colab.install('osmnx')
colab.install('osmnet')

colab.upgrade('geopy')

Introduction#

Google brought with maps the world to our screens, including accurate geocoding and routing for several modalities.

For the most, the usage of maps is interactive. As data and analytics professionals we often need a programmatically support for the services that maps offer us. Preferably free.

It also offers a plethora of development support, but unfortunately most is paid. That is even more so for maps.

The same Google also offers us colab and that is exactly where we are now. Colab is free to use, although it includes a paid pro version.

This notebook runs on the free version of colab and illustrates free packages on the python ecosystem.

Alongside with free code the open source community also offers us a lot of free data and free knowledge. This notebook, and presentation, is just a skin-deep introduction.

Some heroes#

Geoff Boeing is a true leader in demystifying urban data analytics, with a strong emphasis on street networks. His peer reviewed publications are open and accompanied by usable demonstrations using his own OSMnx package. Professor Peter Sanders, see also his Wikipedia page, has moved his interests to other areas but his route planning project shaped the world of truly scalable road routing algorithms. From his alumni I distinguish two persons:

  • Dominik Schultes who won the DIMACS challenge on shortest paths and made it to the Scientific American top 50. Before Dominik’s research scalable shortest paths on large national road networks where heuristics, now they are exact and can be computed at world scale.

  • Dennis Luxen for creating https://github.com/Project-OSRM/osrm-backend which offers a free, scalable, implementation of contraction hierarchies.

Finally, I mention Fletcher Foti who gave us pandana.

Whenever we need more packages, we just install those into our session.

def locate_geopy(description):
    location = FreeLocator().geocode(description)
    if location is not None:
        return location.latitude, location.longitude
    return None, None
